Intel SGX for Ubuntu

Zane Wang2024-11-302024-11-30

不能用 VMWare 虚拟机环境下的 Ubuntu，不支持硬件的 Intel SGX。对于低于 20.04 版本的，我不知道是不是可以安装成功。

（1）下面的操作最好都是在 root 用户权限下操作，避免一些错误。

（2）最好是挂个梯子来操作，不然 git wget 这些可能下载不下来一些库。

a. 到https://github.com/chwzx/Clash-for-linux的Releases里面下载 Clash for Linux 的压缩包，然后解压。在文件夹路径中执行 ./cfs --no-sandbox启动 Clash，然后自己导入节点。

b. 系统设置中网络代理那里配置一下。
